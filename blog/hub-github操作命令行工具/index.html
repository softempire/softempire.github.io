02 Dec 2019
@dingsu

Git现在是我每天用的最多的软件了，跟Visual Studio一样，每天都在用，时时刻刻都在用。Github也是每天都打交道的网站，commit，pr，review，merge，search，每天做这些。

Hub(https://hub.github.com/)就是一款Github的命令行操作工具，支持普通的Github跟Github Enterprise，非常的方便，很多操作比在网页上操作要方便的多。

Hub也支持大部分的Git命令，管网上推荐用Hub来代替Git的功能，同时又提供了Github的操作功能。
